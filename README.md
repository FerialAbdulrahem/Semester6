## ⚙️ Computer System Architecture – Package 4 Simulator
**Course:** CSEN 601 – Computer System Architecture  
**Languages/Tools:** C Programming  

📖 **Description:**  
Implemented **Package 4: Double McHarvard** processor simulator. The project focused on instruction decoding, fetch/decode/execute cycle, and pipeline stages.  

📄 [Read Project Description (PDF)](https://github.com/FerialAbdulrahem/Semester6/blob/main/S6Projects/ComputerSystemArchitecture/CSEN601%20Project%20Description.pdf)

✅ **Outcome:** A working C-based simulator capable of executing instructions with correct cycle counts, register updates, and pipeline behavior.

## 🖥️ Operating System – OSSimulator
**Course:** CSEN 602 – Operating Systems  
**Languages/Tools:** Java  

📖 **Description:**  
Developed an **Operating System Simulator** that models process scheduling, memory management, and system calls. The project required implementing multiple schedulers (Round Robin, HRRN, MLFQ), process control blocks, and synchronization primitives.  

📄 [Read Project Description (PDF)](https://github.com/FerialAbdulrahem/Semester6/blob/main/S6Projects/OperatingSystem/Project%20description.pdf)  

✅ **Outcome:** A complete Java-based OSSimulator capable of running multiple programs, handling scheduling policies, and simulating system calls.  
📎 [Go to Project Folder](https://github.com/FerialAbdulrahem/Semester6/tree/main/S6Projects/OperatingSystem/OSSimulator)
